  • ห้ามนำโพยข้อสอบเข้าห้องสอบ
  • ห้ามแต่งกายผิดระเบียบ
  • ห้ามกินส้มตำปูปลาร้า
  • ห้ามนอนดึก

1.แท็กคำสั่งใดที่ใช้ในการตีเส่นแนวนอนในเอกสาร

  1. p
  2. wbr
  3. hr
  4. nobr

2.ในการสร้างรายการแบบเรียงลำดับข้อใดถูกต้อง

  1. ใช้แท็กคำสั่ง ol และ li
  2. ใช้แท็กคำสั่ง ol และ li
  3. ใช้แท็กคำสั่ง ul และ ui
  4. ใช้แท็กคำสั่ง ul และ li

3.ในการสร้างรายการแบบไม่เรียงลำดับข้อใดถูกต้อง

  1. ใช้แท็กคำสั่ง ol และ li
  2. ใช้แท็กคำสั่ง ol และ oi
  3. ใช้แท็กคำสั่ง ul และ ui
  4. ใช้แท็กคำสั่ง ul และ li

4.คำสั่งที่ทำให้พื้นเอกสารเป็นสีเหลือง

  1. font body color="yellow"
  2. font body color="#fff000"
  3. body bgcolor="yellow"
  4. ถูกทุกข้อ
